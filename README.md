# Tic Tac Toe

## What's this?

A full stack tic tac toe multiplayer game

## Built with

- Next js
- Socket io
- Node js
- Express
- Tailwind Css
- React Hot Toast
- React Confetti

## What's Inside?

- A tic tac toe game
- A chat system
- A notification system
- Sounds when interactions occur
- Uses query parameters for invitation links
